Dataset Summary

The dataset I've chosen consists of alcohol consumption across 197 countries worldwide (more information available on my Project Dataset page). Each country has data on consumption of beer, wine, spirits, "other," and "all types." Data points are available (on most countries) for every year from 2010-2013.

Per my project sketch, I plan to include three panels:

  1. A world map, to directly select or reference each country's annual statistics.
  2. A bubble chart, displaying each country's alcohol consumption. The chart would be animated to show changes year over year, with regions/continents grouped together by color.
  3. Small multiples, showing alcohol consumption (filtered by type) across all countries. This might show all 197 countries, or I may break this out by region/continent as well.
For functionality, I'd like the page set up in a way that would allow users to click on a country, and quickly compare it to other nearby countries.
