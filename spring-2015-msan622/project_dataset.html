Project Dataset: Alcohol Consumption

Jeff Baker

Source

My project dataset is sourced from the World Health Organization (WHO), and consists of recorded alcohol consumption (per capita) by country, from 2000-2013. The data is publicly available for distribution if used for private use or research.

Data

The data consists of a mix of quantitative and categorical data. Data fields include:

  • Year (nominal) - 2000 through 2013
  • Country (categorical) - 197 countries worldwide
  • Beverage Type (categorical) - Consumption grouped by: beer, wine, spirits, other, all types
  • Consumption (continuous) - Recorded alcohol per capita (APC), measured in liters1
After looking at a number of datasets, including NFL season data and advertising data, I chose this one because of it's relative simplicity. I also thought the topic of alcohol is interesting, and comparing how consumption varies by country or continent would be fascinating. Between the categorical and time series components to this dataset, I have a number of visualization techniques to choose from.

This data—reformatted for use in this project—can be found on my repository here.

Dataset Preparation

The formatting of the data is "wide," with the years of data as columns, with rows divided by country and alcohol category. Data was reformatted in R, leveraging the melt function within the reshape2 package. This allowed me to convert the data into a "long" form instead, with the data falling into the four columns listed above. The data includes 0, null, and "No data" values, which will be handled during the data visualization step.
